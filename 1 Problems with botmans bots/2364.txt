Use Notepad to open the HPB_bot.cfg file in the Half-Life\tfc folder.  Add (or remove) "addbot" commands (without the double quotes) at the bottom of the file.  Then save the HPB_bot.cfg file, start Half-Life, select TFC as a Custom game, select Multiplayer, then LAN game, then Create game, then pick a map and click on Ok.

If you have a limited amount of system RAM you may need to create a shortcut to hl.exe and add a parameter to allocate more memory so that you can add more bots (some machines are limited by default to 4 bots unless you change this parameter).  See the 3rd question in the FAQ on the Phineas Bot web site for more details...

http://phineas.valveworld.com/faq.html
