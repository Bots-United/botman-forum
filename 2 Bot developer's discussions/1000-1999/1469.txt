Like Pikachu said, you can build the Full SDK or the Standard SDK with the free Borland command line compiler, but be aware that you can't build the client.dll file unless you have Microsoft Visual C++ (because of the VGUI library).

If you are just playing around with tutorials or simple modifications to the server DLL file (hl.dll or mp.dll) then Borland will work fine.  If you are planning on creating a MOD with a custom HUD layout then you need to buy Microsoft Visual C++.
